== Installation ==

1. Upload 'psp-delays' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit or view any project on the front or backend, you'll be able to add, edit and see delays
4. Delays will automatically adjust the end date based on the number of days delayed
